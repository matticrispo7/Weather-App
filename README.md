# Weather-App
Android App written in Java

## Table of contents
* [Functionalities](#functionalities)
* [Technologies](#technologies)
* [Setup](#setup)

## Functionalities
* Storage Options: File with JSON Format
* HTTP Client
* OpenWeather API
	
## Technologies
* The user can search a City and view Weather Information
* Weather information should be retrieved on-demand through an HTTP Call (with AsyncTask)
* The user can save a City as Favourite
* When the user retrieve the Weather info for a Favourite City the data must be stored and showed in a dedicated History View
	
## Setup
Create a free account on https://openweathermap.org/api and select "Current Weather Data".
Insert API key in MainActivity.java.
